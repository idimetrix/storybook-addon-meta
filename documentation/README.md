## storybook-addon-meta

#### Meta information for Storybook components

## Installation

Install with [npm](https://www.npmjs.com):

```sh
$ npm install --save storybook-addon-meta
```

Install with [yarn](https://yarnpkg.com):

```sh
$ yarn add storybook-addon-meta
```
